# 니짐내짐 홈페이지 (독립 정적 프로젝트)

헬스장 찾을 땐, 니짐내짐 — 회원(B2C)·사장님(B2B) 양면 랜딩 페이지.
외부 개발자가 그대로 실서버에 올릴 수 있도록 **의존성 없는 순수 정적(HTML/CSS/JS)** 으로 구성했습니다.

## 파일 구조
```
site/
├── index.html          회원용 홈 (히어로 + 3동사 + 양면 브릿지 + 사장님 티저 + 앱 다운로드)
├── enter.html          사장님 입점 (히어로 + IoT 시설 원격제어 + 운영 이점 + 입점 신청)
├── assets/
│   ├── css/styles.css  공용 스타일 (CSS 변수로 팔레트 관리)
│   ├── js/main.js      스크롤 진입 애니메이션 (IntersectionObserver, 순수 JS)
│   └── img/            로고 SVG (wordmark/symbol, black/white)
└── .nojekyll           GitHub Pages Jekyll 처리 비활성
```

## 실행 (빌드 불필요)
정적 파일이라 서버에 파일째 올리면 끝입니다. 로컬 확인:
```bash
cd site && python3 -m http.server 8080   # → http://localhost:8080
```

## 브랜드 규칙 (수정 시 유지할 것)
- **팔레트**: Rose `#FF0065` · Sky `#39B4FF` · Ink `#16161C` (audit 실측). `styles.css :root` 변수로 일괄 관리.
- **폰트**: Pretendard (CDN). 오프라인/사내망 배포 시 폰트 로컬 번들 권장.
- **마스터 태그라인** "헬스장 찾을 땐, 니짐내짐" = 로고 락업 옆 / 푸터 / 메타에만. **사장님 페이지 본문·히어로엔 넣지 않음**(운영·회원 언어만).
- **신뢰지표(TRUST BAR)**: 데이터 미확정으로 **의도적으로 제외**된 상태. 수치 확정 후 추가 예정.

## 확정 카피 (변경 금지 — 마케팅/브랜딩 승인본)
- 회원 히어로 H1: **내 주변 헬스장, 최저가로 바로 시작**
- 회원 히어로 서브: **찾고 · 구독하고 · 넘기고 — 헬스장의 처음부터 끝까지 앱 하나로**
- 회원 CTA: **내 주변 헬스장 찾기** / **앱 다운로드**
- 사장님 히어로 H1: **센터 전체를 폰 하나로, 사장님은 회원에만 집중**
